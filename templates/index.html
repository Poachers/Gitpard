Что мы за проект?

Сервис Gitpard обеспечивает анализ git репозиториев с python кодом. Анализ представляется в виде графиков. Тут должно быть побольше инфы!

Анализ

Анализ файлов позваляет просмотреть все файла репозитория. Дата создания строки, её автор и "время жизни" строки.

Отчет

Формирование отчетов по авторам и по файлам. Возможность скачать отчет для дальнейшего анализа.
